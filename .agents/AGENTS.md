# Rules

1. When modifying Japanese study notes, **NEVER** remove existing textbook references or page numbers (e.g., `Genki 2 p164`). Even if these exact page numbers cannot be verified in supplementary lecture slide PDFs, they should be retained as valuable context unless the user explicitly requests their removal. However, **if you CAN verify the exact textbook page number from the provided lecture slide PDFs**, you are allowed to *edit* the page number to the correct one, but do not delete the reference entirely.
